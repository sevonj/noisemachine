Plan for a computer controlled analog synthesizer. This project is in very early steps.  
The design is ~~mostly stolen from~~ slightly inspired by the Prophet-5 synthesizer.

The plan is to design a self contained voice unit that be commanded by a Raspberry Pi, or such.
